Explore: MakeCode

Objective: Complete a few advanced MakeCode tutorials to get a sense for various capabilites and code complexity and efficiencies.

  1. Creative Projects

The First Steps tutorial ends by recommending three creative projects to explore. The first two need 2 micro:bit devices to complete.

  • Proximity Beacon - alert when devices are in proximity.
  • Pressure Switch Alarm - alert when user steps on pressure pad.
  • Reaction Game - create a game with DIY switches as inputs.

We'll try out the third project (with one micro:bit) first before going on to explore the MakeCode site for more tutorials.
